Команда:
int damage;		    // Урон
int healing;	            // Лечение
int type;           	    // Тип комманды
bool owner;     	    // Принадлежность
string name;	            // Наименование комманды

Типы:
0. Normal, нормальный.
1. Fire, огонь.
2. Water, вода.
3. Grass, зелень.
4. Electric, электрический
5. Ice, лед.
6. Fighting, боевой.
7. Poison, яд
8. Ground, земля
9. Flying, летающий
10. Physchic, психический. 
11. Bug, жук.
12. Rock, камень.
13. Ggost, призрачный.
14. Dragon, драконний.
15. Dark, темный.
16. Steel, сталь.
17. Fairy, фея. 
Таблица слабостей находится в папке.
Дополнительно: Феи сильны против Боевого, Драконнего и Темного, но
слабы перед Ядом, Призрачным и Сталью.
Белый: урон не изменяется.
Красный: урон в 2 раза скоращается.
Черный: 0 ед. урона
Зеленый: урон в 2 раза увеличивается.


Уровень всех ПокеМонов по умолчанию 30.
